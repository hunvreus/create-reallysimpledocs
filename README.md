# create-reallysimpledocs

Scaffold a new [ReallySimpleDocs](https://github.com/hunvreus/reallysimpledocs) site.

## Usage

```bash
npm create reallysimpledocs@latest my-docs
# or
npx create-reallysimpledocs@latest my-docs
```

## Options

- `--force`: copy into a non-empty directory.
- `--repo <owner/name>`: GitHub repo to clone.
- `--ref <ref>`: Git ref (tag/branch/commit). Defaults to `HEAD`.

## Local development

```bash
node bin/create-reallysimpledocs.js my-docs --force
```
